# scPRISM

scPRISM是一个旨在通过采样过程分析来还原真实表达谱信号的即插即用方法。

## 优点

prism分析框架基于真实的测序物理过程建模，其中最具有随机性的步骤是，测序过程等价于一个伯努利采样过程，这一步引入了巨大的随机性且平滑化了曲线导致曲线形态趋于一致。

本方法致力于系统性的建模伯努利采样的过程，量化该过程引入的随机性（每一个观测量的置信度），并提供更好的信号（更具备线性可分性）

此外，本框架的一大特点是，能够原生的缓解甚至防止批次效应，批次效应常常被建模为一个固定的乘数缩放因子，主要作用于测序深度上，我们的框架本身能够很好的解耦测序深度对真表达值i的影响，在多种分析中，我们提取出的信号与测序深度的相关性接近0，远低于原始umi计数或者常用的normalize total+log1p（lognoraml方法也会存在较高的相关性，可能是因为测序深度会改变伯努利采样的均值）

基于prism拟合的基因的Fg曲线，呈现出很强的基因间特异性，且随机划分到更小的样本中拟合，同样呈现出极强曲线形态的一致性，说明Fg的拟合对于小样本都已经足够精准，大样本的Fg拟合与小样本的Fg拟合结果一致，说明基因表达的底层结果是噪声平滑后的曲线（因此更多的样本拟合Fg并没有带来更复杂的曲线形态）我们的框架可以被证明是凹优化的，因此并不用担心局部最优的影响。

在批次效应上，prism框架对逐个批次进行拟合，不同的批次呈现出高一致性的曲线形态。而少数批次曲线可能存在较大的偏差，这可以作为qc的手段之一。而这种偏差性质在基因间出现一定的一致性，也就是说这种batch的质量问题，在基因间具有一定的一致性的，的确可以作为潜在qc质控的手段。
此外，不同批次的Fg拟合结果都具备高度的一致性，因此，每个批次单独拟合Fg，提取信号，可以作为一种跨批次的信号归一化手段。我们的信号能够更加还原真实的表达量，且具备和原始计数接近的信息量，并且效果要比lognormal更好。
