# scPRISM：面向单细胞分子的概率独立信号提取

---

## 1. 符号与生成模型

### 1.1 符号表

| 符号 | 名称 | 定义 | 备注 |
|---|---|---|---|
| $g$ | 基因索引 | $g = 1, \ldots, G$ | $G$ 为选定建模基因集合 $\mathcal{G}$ 的大小 |
| $c$ | 细胞索引 | $c = 1, \ldots, C$ | |
| $p_{gc} \in (0,1)$ | 表达比率 | 细胞 $c$ 中基因 $g$ 占总分子池的比率 | 直接拟合目标；相对于基因集合 $\mathcal{G}$ 定义 |
| $S > 0$ | 标尺超参数 | 用户指定的缩放常数 | 纯量纲转换；不改变 $F_g$ 的形状，只控制网格的绝对尺度 |
| $\mu_{gc}$ | 缩放表达量 | $\mu_{gc} = p_{gc} \cdot S$ | 为 $p_{gc}$ 赋予更直观的"分子数"量纲的等价表示 |
| $\epsilon_c > 0$ | 细胞效率因子 | 细胞 $c$ 的全局乘性偏置 | 对所有基因共享；来源于捕获效率、透化效率、液滴产率等的乘积；不假设任何参数分布 |
| $n_{gc}$ | 真实分子数 | 细胞 $c$ 中基因 $g$ 的实际mRNA分子数 | 不可观测 |
| $T_c$ | 总分子数 | $T_c = \sum_{g \in \mathcal{G}} n_{gc}$ | 不可观测；$\mathcal{G}$ 内的总分子数 |
| $r \in (0,1)$ | 捕获效率 | 每个分子被捕获为 UMI 的概率 | 不可观测；在条件化后不出现于模型中 |
| $X_{gc}$ | 观测 UMI 计数 | 基因 $g$ 在细胞 $c$ 中的最终观测值 | |
| $N_c$ | 总 UMI 计数 | $N_c = \sum_{g \in \mathcal{G}} X_{gc}$ | 相对于基因集合 $\mathcal{G}$ 定义 |
| $F_g$ | 基因先验 | $p_{gc} \sim F_g$ | 基因 $g$ 表达比率的群体级有效先验（effective prior） |
